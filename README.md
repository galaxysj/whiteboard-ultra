# Whiteboard Ultra - A whiteboard with features others don't have<br>
AI-powered tools, embeddable URLs, and built-in math tools

## introduction
Whiteboard Ultra is a digital whiteboard that supports link embedding, drawing compass, AI Features and more.


## Dependencies
- Node.js >= 20.19.0
- npm
Other Dependencies will going 

## How to run


First, clone the repository.
```bash
git clone https://github.com/galaxysj/whiteboard-ultra
cd whiteboard-ultra
```
Next, install and build the project
```bash
npm install
npm run build
```
Finally, start the app.
```bash
npm run dev
```
Now, go to http://localhost:5173 or http://SERVER-IP:5173 to use the board

## Features
- AI Tools: Insert, Build, and Ask about your board.
- Embed Tools: Embed HTML, images, videos, web and more.
- Math Tools: Use Math Tools like calculator, ruler, petrotactor and graph.
